# Comparacion con el manual de correspondencia

## Funciones equivalentes

- Inicio con menu lateral y tarjetas principales.
- Solicitud de numero de nota.
- Registro de comunicaciones despachadas.
- Registro de comunicaciones recibidas.
- Carga de archivo digital.
- Derivacion de comunicacion recibida.
- Seguimiento por codigo.
- Notas por recibir.
- Reportes de despachadas, recibidas y derivadas por area.

## Mejoras incluidas en la demo

- Login por credenciales y sesion por usuario.
- Administrador para registrar unidades organizacionales y usuarios.
- Restriccion de documentos por unidad para funcionarios.
- Recepcion/Ventanilla como punto inicial para toda documentacion entrante.
- Derivacion controlada antes de que una unidad vea el documento.
- Una sola vista de detalle con expediente, responsable, estado, archivo digital e historial.
- Indicadores visibles para documentos sin archivo digital.
- Confirmacion de recepcion fisica.
- Busqueda global en todas las bandejas.
- Formulario con CITE, referencia, emisor, receptor, anexos, copias, vias, respuesta y externo.
- Base preparada para version SaaS multiempresa.

## Pendiente para version productiva

- Login real con usuarios de dominio o usuario institucional.
- Cifrado de contrasenas, recuperacion segura y bloqueo por intentos.
- Roles diferenciados: usuario, supervisor, administrador y auditor.
- Archivos PDF reales con almacenamiento seguro.
- Exportacion real a PDF y Excel.
- Auditoria inalterable de acciones.
- Parametrizacion de areas, cargos, tipos de nota e instrucciones.
